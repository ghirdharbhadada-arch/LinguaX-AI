# 🌍 LinguaX AI

An advanced Language Translation Tool developed using **HTML, CSS and JavaScript**.

## ✨ Features

- 🌐 Multi-language Translation
- 🎤 Voice Input
- 🔊 Text to Speech
- 📋 Copy Translation
- 📥 Download Translation
- 🌙 Dark / Light Mode
- 📜 Translation History
- 🔄 Language Swap
- 📱 Responsive Design

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- MyMemory Translation API
